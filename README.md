# Student ERP System

A web-based Student ERP system developed using PHP, MySQL, Bootstrap, and JavaScript.

## Features

- Admin Login System
- Dashboard with statistics and charts
- Student Management (Add, Edit, Delete, Search)
- Attendance Management
- Attendance Reports
- Student Profile Page
- Fees Management
- Fees Reports
- Result / Marks Management
- Result Reports

## Technologies Used

- PHP
- MySQL
- Bootstrap
- JavaScript
- Chart.js

## How to Run

1. Install XAMPP
2. Copy project to `htdocs`
3. Start Apache and MySQL
4. Import the database in phpMyAdmin
5. Open browser and go to:

http://localhost/student_management
